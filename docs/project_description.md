# 项目说明

## 项目名称

信息学院学生学习导航平台

## 项目背景

信息学院学生在学习专业课程和完成实验项目时，通常需要同时查找课程资料、实验要求、开发工具、项目案例和报告材料。不同类型的信息分散在课件、网页、代码仓库和文档中，查找过程容易影响学习效率。为解决这一问题，本项目设计并实现一个静态网站形式的学习导航平台，用统一的页面结构集中展示相关学习内容。

## 选题原因

本项目选题贴近信息学院课程实践场景，能够综合体现 Web 前端开发、页面布局设计、JavaScript 基础交互、项目文档整理和运行展示等能力。静态网站不需要后端环境和数据库，适合期末综合项目展示，也便于在浏览器、VS Code Live Server 和 GitHub Pages 等方式下运行。

## 设计思路

网站采用“首页总览 + 分类页面展示”的结构。首页负责展示平台主题、功能入口和公告信息；课程资源页、实验作业页、项目展示页、学习工具页分别对应不同学习任务；关于我们页说明项目目的、合规原则、数据来源和学习诚信要求。

视觉设计采用蓝白色校园风格，整体保持简洁、正式、清晰，适合实验报告截图。页面使用卡片式布局组织信息，并通过统一导航栏、统一按钮样式、统一页脚增强项目完整性。

## 功能模块

1. 首页模块：展示网站标题、导航栏、首页横幅、平台简介、主要功能入口、公告区域和页脚。
2. 课程资源模块：展示 Web 开发、软件工程、机器学习、数字影像技术、创新实验等课程资源卡片。
3. 实验作业模块：展示实验一至实验五任务列表，突出实验五期末大作业。
4. 项目展示模块：展示静态网站、机器学习、小程序、图像处理、文档管理等项目案例。
5. 学习工具模块：展示 VS Code、GitHub、MDN Web Docs、菜鸟教程、ProcessOn、Draw.io、W3C Validator、Markdown Guide、Mermaid Live Editor 等工具链接。
6. 关于说明模块：说明项目设计目的、选题原因、功能依据、合规说明、隐私与数据来源说明、学习诚信说明。
7. JavaScript 交互模块：实现导航高亮、搜索筛选、公告切换、返回顶部、卡片淡入效果。

## 技术路线

项目采用纯前端静态网站技术路线：

- HTML：负责页面结构和语义化内容组织。
- CSS：负责蓝白色校园风格、卡片布局、响应式设计和视觉效果。
- JavaScript：负责导航高亮、搜索筛选、公告切换、返回顶部和淡入动画。
- Markdown：负责 README 和实验报告材料整理。

项目不使用 Vue、React、Node.js、数据库或后端服务，降低运行门槛，保证项目可以直接通过浏览器打开。

## 创新点

1. 将课程资源、实验任务、项目案例、学习工具和项目说明集中到一个学习导航平台中。
2. 使用统一卡片化视觉风格，提高页面信息可读性和报告展示效果。
3. 在课程资源页和实验作业页提供关键词筛选功能，提升页面可交互性。
4. 首页公告区域支持自动切换和手动切换，增强动态展示效果。
5. 文档材料与网站代码同步整理，便于课程报告撰写、截图展示和仓库提交。

## 合规说明

本项目内容围绕课程学习、专业实践和工具导航展开，不包含违法违规、不良导向或侵犯他人权益的内容。项目不设置登录注册功能，不连接数据库，不采集个人隐私信息。页面中的项目案例和学习内容均为课程学习场景示例，外部链接仅作为公开学习资源入口。
