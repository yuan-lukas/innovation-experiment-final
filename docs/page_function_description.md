# 页面元素与功能说明

## 首页 index.html

首页包含统一导航栏、首页横幅、平台简介、主要功能入口、公告区域和页脚。导航栏用于进入课程资源、实验作业、项目展示、学习工具和关于我们页面。首页横幅展示网站名称和核心定位，突出“信息学院学生学习导航平台”的主题。平台简介区域说明项目用途，数据统计卡片展示页面数量、课程模块、实验任务和隐私数据采集情况。主要功能入口使用四张卡片分别跳转到课程资源、实验作业、项目展示和学习工具页面。公告区域提供三条课程项目相关提示，并支持自动切换和圆点手动切换。

## 课程资源页 pages/courses.html

课程资源页用于展示信息学院常见课程资源。页面顶部为页面横幅，说明课程资源页用途。主体区域提供搜索框，用户可以输入课程名称、技术关键词或学习方向进行筛选。课程卡片包括 Web 开发、软件工程、机器学习、数字影像技术、创新实验。每张卡片包含课程名称、课程简介、学习重点和查看详情按钮。页面下方设置课程学习建议详情区，补充每门课程的推荐练习、项目关联和参考入口。

## 实验作业页 pages/experiments.html

实验作业页用于梳理实验一到实验五的任务内容。页面提供实验搜索框，可根据实验名称、技术关键词或任务关键词筛选卡片。实验列表采用时间线式卡片布局，每项实验包含编号、完成状态、实验名称、任务说明、任务要点和技术标签。实验五期末大作业使用重点样式突出显示。页面下方增加实验五验收要点，从页面完整性、交互可用性、文档规范性和展示截图四个方面辅助检查。

## 项目展示页 pages/projects.html

项目展示页用于展示学生创新项目类型。页面采用项目卡片布局，包含静态网站项目、机器学习项目、小程序项目、图像处理项目、文档管理项目和课程项目管理实践。每个项目卡片包含项目类型视觉区、项目名称、项目简介、技术标签和查看详情按钮。页面下方设置项目详情说明，补充每个项目的功能构想、技术路线和可展示成果，使项目展示更加完整。

## 学习工具页 pages/tools.html

学习工具页用于集中展示课程项目开发常用工具和网站链接。页面包含 VS Code、GitHub、MDN Web Docs、菜鸟教程、ProcessOn、Draw.io、W3C Validator、Markdown Guide、Mermaid Live Editor 等工具卡片。每张卡片包含工具标识、工具名称、用途说明、适用场景和外部链接按钮，方便用户快速进入学习资源网站。

## 关于我们页 pages/about.html

关于我们页用于说明项目建设背景和规范要求。页面包含项目设计目的、选题原因、功能设置依据、完成说明、内容合规说明、隐私与数据来源说明、学习诚信说明等内容。该页面适合放入实验报告中，用于解释项目主题、功能选择和合规处理。

## 统一页面功能

所有页面均使用统一导航栏、统一页脚、统一按钮样式和统一卡片布局。JavaScript 会根据当前页面自动高亮导航菜单。页面右下角提供返回顶部按钮，滚动一定距离后显示。卡片元素加载后具有淡入效果，提升页面展示体验。移动端宽度下导航栏会折叠为按钮，卡片布局自动变为单列或双列，保证手机端正常浏览。
