# 信息学院学生学习导航平台

《创新实验》课程期末综合项目——面向信息学院学生学习场景的轻量静态导航网站。

## 项目简介

本平台无需后端数据库，无需登录注册功能，直接用浏览器即可打开运行。旨在集中展示课程资源、实验作业、创新项目、学习工具和项目说明，帮助学生快速定位学习资料。

## 技术栈

| 技术 | 用途 |
|------|------|
| HTML5 | 页面结构与内容组织 |
| CSS3 | 统一视觉风格、卡片布局、响应式设计 |
| JavaScript | 导航高亮、搜索筛选、公告切换、返回顶部、卡片淡入效果 |
| Markdown | 项目说明和实验报告材料整理（docs/ 目录） |

## 页面功能说明

| 页面 | 说明 |
|------|------|
| 首页 | 网站标题、导航栏、横幅、平台简介、功能入口、公告区域、页脚 |
| 课程资源 | 展示 Web 开发、软件工程、机器学习等课程卡片，支持关键词搜索 |
| 实验作业 | 展示实验一到实验五任务列表，突出实验五期末大作业 |
| 项目展示 | 展示静态网站、机器学习、小程序、图像处理等项目卡片 |
| 学习工具 | 提供 VS Code、GitHub、MDN、菜鸟教程、ProcessOn 等工具链接 |
| 关于我们 | 项目说明、成员信息、合规说明、隐私与数据来源、学习诚信说明 |

## 运行方式

- **方式一**：直接双击 `index.html` 用浏览器打开
- **方式二**：使用 VS Code Live Server 插件运行（推荐）
