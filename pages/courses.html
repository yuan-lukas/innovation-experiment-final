Web 开发

建议按照“结构、样式、交互、适配、测试”的顺序学习。完成本网站时主要使用 HTML 搭建页面结构，使用 CSS 控制卡片布局和响应式效果，使用 JavaScript 实现搜索和公告切换。

  • 推荐练习：制作个人课程导航页、实验任务页和工具链接页。
  • 项目关联：本项目 6 个页面均体现 Web 开发综合应用。
  • 参考入口：MDN Web Docs、菜鸟教程、浏览器开发者工具。

机器学习

机器学习课程重点在于理解数据、特征、模型和评估。本网站不实现模型训练，但在项目展示页中保留机器学习项目案例，便于说明信息学院项目方向的多样性。

  • 推荐练习：用模拟学习数据完成分类或预测任务。
  • 项目关联：项目展示页中的“学习状态预测项目”可作为机器学习方向示例。
  • 参考入口：Python、Scikit-learn、Jupyter Notebook。

数字影像技术

数字影像技术适合与图像处理项目结合，常见内容包括图像读取、颜色空间转换、滤波增强、边缘检测和结果展示。

  • 推荐练习：比较原图、灰度图、增强图和边缘检测图。
  • 项目关联：项目展示页中的“图像增强与识别项目”可作为图像处理方向示例。
  • 参考入口：OpenCV、图像格式基础、可视化报告。

创新实验
